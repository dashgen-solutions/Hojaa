# Changelog

All notable changes to Hojaa will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22

### Added
- AI-powered progressive requirements discovery (10 targeted questions per session)
- Document upload and analysis (PDF, DOCX, TXT)
- Interactive requirement tree visualization with React Flow
- Per-feature contextual chat exploration
- Multi-source ingestion (meeting notes, emails, Slack threads)
- Scope change detection with source attribution
- Audit trail with full change history
- Time-travel view (browse tree at any historical point)
- Planning board (Kanban: Backlog, TODO, In Progress, Review, Done)
- AI-generated acceptance criteria for planning cards
- Export to PDF, JSON, and Markdown
- Jira and Slack integrations
- WebSocket-based real-time collaboration with presence indicators
- Multi-LLM support (OpenAI, Anthropic, Azure OpenAI, Ollama)
- Per-task model routing and cost-tier optimization
- Role-based access control with organizations
- White-labeling / branding customization
- Email notifications via SMTP
- Docker Compose deployment (PostgreSQL + FastAPI + Next.js)
- Comprehensive API documentation (Swagger UI + ReDoc)
